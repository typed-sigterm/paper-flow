试卷扫描件转 Word，但是专业版。

## 功能

- 扫描件转 Word
- 自动切割题目、修复格式
- 去水印 / 重绘插图

## 安装

PaperFlow 不提供在线服务，需要自行部署，教程见 [DEPLOYMENT.md](./DEPLOYMENT.md)。
